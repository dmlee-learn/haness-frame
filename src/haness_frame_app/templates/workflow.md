## 1. Search

Project Scout searches for related products, open source examples, common
problems, and implementation patterns.

## 2. Discuss

Planner, Designer, Architect, Critic, and Decision Maker review the evidence
and decide the design.

## 3. Develop

Coder implements only after the decision record contains an accepted plan.

## 4. Review Loop

If the implementation fails, feed the failure back into the discussion stage
before patching again.
